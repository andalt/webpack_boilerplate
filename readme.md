# Webpack сборка

Сборка, основанная на Webpack

# Команды

### DevServer

Запускает девсервер, локальная папка build не существует
```sh
npm start
```

### Build

```sh
npm run build
```

Перед сборкой очищает папку build. Собирает без минификаций.

### Production build

Перед сборкой очищает папку build. Собирает с минификацией.

```sh
npm run build:prod
```
